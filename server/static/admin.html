아직 등록된 상태 프리셋가 없습니다.

🍎 제어 센터용 "단축어" 만드는 법 (초간단)

단축어 앱에서 "E-ink 갱신"이라는 단축어를 하나만 직접 만들어두면, 제어 센터에서 클릭 한 번에 모든 잉크 화면을 고를 수 있습니다!

  1. 단축어 앱을 열고 새 단축어를 만듭니다.
  2. URL 동작을 추가하고 아래 주소를 넣습니다.
    http://your-server-ip/api/shortcuts/list
  3. URL의 콘텐츠 가져오기 동작을 이어서 추가합니다.
  4. 목록에서 선택 동작을 이어서 추가합니다. (이제 '프리셋 이름'만 예쁘게 나옵니다)
  5. URL 동작을 하나 더 추가하고 아래 주소를 넣습니다. (가운데에 '선택한 항목' 변수를 바로 넣으세요)
    http://your-server-ip/api/presets/(선택한 항목)/activate
  6. URL의 콘텐츠 가져오기 동작을 마지막으로 추가하고, 화살표(더 보기)를 눌러 방법을 POST로 바꿉니다.

이 단축어를 제어 센터나 메뉴 막대에 등록하시면, 이제부터 프리셋이 추가될 때마다 알아서 목록이 갱신됩니다!

⚙️ 서버 설정 ▸
Apple 단축어가 이 URL로 요청을 보냅니다. 외부에서 접근 가능한 IP 또는 도메인을 입력하세요.
